( About )

La Transfo Code Source « open source » * le protocole de création de laboratoires d’innovation internes aux collectivités territoriales, expérimenté lors du programme La Transfo entre 2016 et 2020. Sept principes actifs composent La Transfo: poser un cadre expérimental (ce), passer en mode pluridisciplinaire (mp), rythmer le programme (p), impliquer et former les agents (a), travailler à partir de cas concrets (cc), s’appuyer sur l'écosystème (e), se projeter dans le futur labo (fl). Ce site vise à expliciter ces principes et apporte un retour d’expérience sur leur mise en oeuvre dans les différentes collectivités engagées dans La Transfo. L’objectif est de permettre à chacun de se réapproprier et d’ajuster ces principes d’action à son contexte.

* En informatique, le code source est un texte qui présente les instructions composant un programme sous une forme lisible, telles qu'elles ont été écrites dans un langage de programmation.
Dans la communauté open source, les développeurs mettent gratuitement le code source à disposition d’autres programmeurs pour inciter à la réutilisation de briques logicielles développées ailleurs et ainsi développer ou améliorer des logiciels à long terme.
